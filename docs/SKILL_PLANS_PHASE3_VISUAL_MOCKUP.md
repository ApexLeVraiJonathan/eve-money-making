## Color & Style Reference

### Status Colors
- **Prerequisite skills**: `bg-muted/40` with `â¤·` icon
- **New skills** (just added): `animate-highlight` (fades in/out)
- **Cannot move**: `opacity-40` on disabled buttons
- **In plan** (skill browser): `text-muted-foreground` with checkmark

### Typography
- **Skill names**: `font-medium text-[13px]`
- **Levels**: `font-mono text-[10px]` in badges
- **Training time**: `font-mono text-[11px] text-muted-foreground`
- **Section headers**: `text-sm font-semibold`

### Spacing
- **Header cards**: `gap-4` between them
- **Form sections**: `space-y-6` for major sections
- **Table padding**: `px-2 py-1.5` for cells
- **Button gaps**: `gap-2` for groups, `gap-1` for tight groups
